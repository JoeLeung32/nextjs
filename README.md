1. [Manual Setup Next.js](./readme/STEP1_ManualSetupNextjs.md)

## Update Machine
- Upgrade Node.js
```text
sudo npm cache clean -f
sudo npm install -g n
sudo n stable
```
- Upgrade Yarn
```text
yarn policies set-version
```
- Upgrade npm
```text
npm install -g npm@latest
```
- Upgrade Git
```text
brew install git
```
